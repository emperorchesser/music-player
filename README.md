# Music Player

A sleek, lightweight web-based music player designed for users who prefer local file management and high-performance playback to potentially annoying online streaming services.

## 🚀 Features

* **Spotify-Inspired UI**: A familiar, intuitive layout optimized for desktop browsing.
* **Modern Iconography**: Features high-quality outline icons for a clean and professional aesthetic.
* **Performance-First**: Focused on fast loading and minimal resource consumption.

## 🛠️ Built With

* **HTML, CSS and JavaScript**: Custom layout inspired by modern streaming interfaces.
* **Font Awesome Pro**: Enhanced iconography provided via Font Awesome 7 Pro.
* **Google AI Studio**: Core logic and architectural scaffolding assisted by the company's AI models.
