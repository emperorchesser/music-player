# Suonation • In Development

A sleek, lightweight web-based music player built for people who want to truly take control of their music. If you are tired of subscription fees, Internet dropouts, and bloated apps, Suonation is a solid choice for you. It offers a premium, high-performance playback experience using your own local files, all wrapped in a familiar interface.

## Core Features

* 🎨 **Dynamic Aesthetic**: The app's background gradients and hover states smoothly adapt in real time to match the color of the currently playing album.
* 🌓 **Light & Dark Mode**: A fully custom-themed UI that respects your eyes.
* 🔍 **Smart Search Engine (Beta)**: A custom-built, typo-tolerant search algorithm that catches spelling mistakes and highlights your top result.
* 🎛️ **Power-User Settings**: Features customizable audio crossfading, explicit content filtering, and on-the-fly audio quality swapping.
* ❤️ **Virtual Playlists**: A "Liked Songs" system that instantly aggregates your favorite tracks into a dedicated playlist.
* 💾 **Persistent Memory**: Uses local storage to remember your settings, your search history, and the exact timestamp of the last song you were listening to.

## ⌨️ Keyboard Shortcuts

Navigate the entire player without ever touching your mouse. Note that **shortcuts are safely disabled while typing in the search bar**.

| Key | Action |
| :--- | :--- |
| <kbd>Space</kbd> | Play / Pause |
| <kbd>Ctrl</kbd> + <kbd>→</kbd> | Next Track |
| <kbd>Ctrl</kbd> + <kbd>←</kbd> | Previous Track |
| <kbd>→</kbd> / <kbd>←</kbd> | Seek forward / backward by 10 seconds |
| <kbd>↑</kbd> / <kbd>↓</kbd> | Increase / Decrease the volume |
| <kbd>M</kbd> | Mute / Unmute the audio |
| <kbd>L</kbd> | Like / Unlike the current song |
| <kbd>R</kbd> | Cycle Repeat modes (Off, All, One) |
| <kbd>Alt</kbd> + <kbd>S</kbd> | Instantly jump to the search box inside the Search tab |

Suonation is also integrated with the **Web Media Session API**, which means your physical keyboard media keys (Play/Pause/Skip) will work globally, and your current song will show up in your operating system's native media overlay.
