# Music Player

A sleek, lightweight web-based music player designed for users who prefer local file management and high-performance playback to online streaming services.

## 🚀 Features
* **Spotify-Inspired UI**: A familiar, intuitive layout optimized for desktop and mobile browsing.
* **Modern Iconography**: Features high-quality outline and duotone icons for a clean, professional aesthetic.
* **Performance-First**: Focused on fast loading and minimal resource consumption.

## 🛠️ Built With
* **HTML5/CSS3**: Custom layout inspired by modern streaming interfaces.
* **Font Awesome Pro**: Enhanced iconography provided via the [rizmyabdulla/fontawesome-pro](https://github.com) distribution.
* **Google AI Studio**: Core logic and architectural scaffolding assisted by Google AI.
